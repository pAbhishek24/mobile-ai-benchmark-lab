# AI Lab Reports

This folder contains human-readable comparison reports derived from benchmark runs in `ai-lab/results/`.

Notes:
- Reports are safe to commit (they should not contain private data).
- Model binaries (`*.gguf`) must never be committed.
- Debug logs under `ai-lab/logs/` are gitignored by default.

## Reports

- `qwen-s24-ultra-comparison.md`: Comparison of Qwen2.5 1.5B vs 0.5B runs on Samsung S24 Ultra.
